# Атлас изобретений

Интерактивная карта изобретений, открытий и технологий человечества — от освоения огня до искусственного интеллекта.

[Открыть сайт на GitHub Pages](https://barzugini.github.io/atlas-izobreteniy/)

## Что внутри

- 12 основных областей
- 48 тематических ветвей
- 192 узла изобретений и открытий
- интерактивные связи между категориями
- адаптивный интерфейс на русском языке

Сейчас проект содержит структуру и навигацию. Подробные карточки отдельных изобретений будут добавляться постепенно.

## Размещение

Сайт хранится в этом репозитории и автоматически публикуется через GitHub Actions и GitHub Pages после изменений в ветке `main`.

## Локальный запуск

Требуется Node.js 22.13 или новее.

```bash
npm ci
npm run dev:next
```

Статическая сборка:

```bash
npm run build:pages
```

Результат создаётся в каталоге `out`.
